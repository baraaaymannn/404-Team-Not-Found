// Core Concepts

المفاهيم الأساسية اللي بيتبنى عليها الـ Database Schema

🗺️
CONCEPT 01 / 04
الـ Schema
هو المخطط اللي بيوضح إزاي الجداول (Tables) متربطة ببعض في قاعدة البيانات.
Blueprint of how tables are structured & related.
📦
CONCEPT 02 / 04
الـ Entity
كل جدول بيمثل كيان (Entity) — زي CUSTOMERS أو PRODUCTS.
Each table = one entity in the real world.
🔑
CONCEPT 03 / 04
الـ Attribute
كل عمود في الجدول بيمثل خاصية (Attribute) — زي الاسم أو السعر.
Columns = attributes / properties of the entity.
🔗
CONCEPT 04 / 04
الـ Relationship
العلاقة بين الجداول بتتعمل عن طريق المفاتيح الأجنبية (Foreign Keys).
Tables connect via Foreign Keys (FK).
